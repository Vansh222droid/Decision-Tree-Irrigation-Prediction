# Decision Tree Irrigation Prediction

## Project Overview

This project uses a Decision Tree Classification algorithm to predict irrigation requirements based on soil, weather, and environmental conditions. The model analyzes multiple agricultural factors and helps determine whether irrigation is needed, supporting efficient water management and smart farming practices.

## Dataset Features

- Soil pH
- Soil Moisture
- Organic Carbon
- Electrical Conductivity
- Temperature (°C)
- Humidity (%)
- Rainfall (mm)
- Sunlight Hours
- Wind Speed (km/h)
- Field Area (hectare)
- Previous Irrigation (mm)

1. Data Collection
2. Data Cleaning

7. Model Evaluation
8. Prediction and Analysis

## Model Used

Decision Tree Classifier

## Evaluation Metrics

- Accuracy Score
- Classification Report
- Confusion Matrix

## Conclusion

The Decision Tree Classification model successfully predicts irrigation requirements using soil and environmental factors. The project demonstrates how machine learning can support precision agriculture by improving irrigation planning, reducing water wastage, and helping farmers make data-driven decisions.

## Key Learning

This project provided hands-on experience in data preprocessing, exploratory data analysis, classification algorithms, model evaluation, and agricultural data analytics using Machine Learning.
